# Colon Cancer Classifiers

Binary classification of **colon cancer vs. diverticulitis** on Dataset109_CC.
Four models are provided: DenseNet3D, MedicalNet (ResNet-18), SwinUNETR, and DINOv2-2D.

---

## Training

All scripts share the same CLI pattern and log to MLflow (`mlruns/`).

```bash
# DenseNet-3D
python train_classifier_densenet3d.py

# MedicalNet ResNet-18
python train_classifier_medicalnet.py

# SwinUNETR
python train_classifier_swinunetr.py --patch_size 96 96 96 --max_epochs 200

# DINOv2 2-D (slice-based)
python train_classifier_dinov2_2d.py
```

---

## Inference

```bash
# DenseNet-3D — full volume, single forward pass
python inference_unified.py --model_type densenet3d \
    --checkpoint checkpoints/densenet3d_best.ckpt --split val

# MedicalNet — full volume
python inference_unified.py --model_type medicalnet \
    --checkpoint checkpoints/medicalnet_best.ckpt --split val

# SwinUNETR — sliding window (96³ patches)
python inference_unified.py --model_type swinunetr \
    --checkpoint checkpoints/swinunetr_best.ckpt --split val \
    --patch_size 96 96 96 --overlap 0.5 --sw_batch_size 4

# DINOv2-2D (meddinov3_2d model type in inference script)
# Requires MEDDINOV3_CKPT_PATH and MEDDINOV3_CODE_PATH env vars (see below)
python inference_unified.py --model_type meddinov3_2d \
    --checkpoint checkpoints/dinov2_2d_best.ckpt --split val
```

Predictions and metrics are saved to `results/` as CSV files.

## MedDINOv3 setup (for `meddinov3_2d` inference)

This model requires the [MedDINOv3](https://arxiv.org/abs/2509.02379) codebase and
its pretrained ViT-B/16 checkpoint. Set the following environment variables:

```bash
export MEDDINOV3_CODE_PATH=/path/to/MedDINOv3/nnUNet/nnunetv2/training/nnUNetTrainer/dinov3
export MEDDINOV3_CKPT_PATH=/path/to/meddinov3_vitb16_ct3m.pth
```

## Data format

The dataset must follow the nnU-Net/MONAI layout generated by `generate_datasetjson.py`.
Run `main_dicom.py` first to convert raw DICOM series to NRRD, then `generate_datasetjson.py`
to produce `dataset.json`.
